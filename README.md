# Common Backend for Prathmesh Roadlines

This is the shared backend server for all Prathmesh Roadlines panels.

## Features
- Authentication and authorization
- Company management
- Vehicle owner management
- Driver management
- Manager management
- Trip management
- Document management
- Payment processing
- Reports and analytics

## Tech Stack
- Node.js with Express.js
- TypeScript
- MongoDB with Mongoose
- JWT for authentication
- Multer for file uploads
- CORS for cross-origin requests
